1- Creamos una carpeta y abrimos le bash 2- Creamos el repositorio local $ git init 3- $ git status , archivos en el area de trabajo 4- Agregarlo al staggin area, con $ git add nombreArchivo con . agrega todos los archivos 5- Mandar archivo al repositorio local , $ git commit -m "mensaje" 6- git remote add origin https://github.com/simongerula/repositorio.git 7- git push -u origin master 8- Para actualizar repetir pasos 4,5 y 7 9- Cuando hacemos proyectos compartidos, solo uno hace el init y el resto hace git clone 10- Settings > Manage access > Invite a collaborator 11- Braches > Add rule > La primer opcion check y dos sub-check 12- El invitado clona el repositorio : $ git clone https://github.com/simongerula/mp_interiorista.git 13- Ver cuantas ramas tiene, git branch 14- $ git branch nombreNuevaRama 15- $ cambiar de rama, $ git checkout nombreNuevaRama 16- Enviar rama a repositorio original $ git push --set-upstream origin nombreNuevaRama 17- Solicitar que una la rama secundaria a la principal , "New pull request" 18- Para conocer las ramas , $ git branch -l 19- Para crear una rama nueva, $ git branch name 20- Cambiar de rama $ git checkout rama_destino
